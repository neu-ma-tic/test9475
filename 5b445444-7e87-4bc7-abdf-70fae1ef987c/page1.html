Documentation

+help

Cette commande vous permet d'avoir la liste complète des commandes de DiscordBot.Js.

+server-info

Cette commande vous permet d'afficher des informations concernant le serveur.

+user-info

Cette commande vous permet d'afficher des informations non personnel concernant un membre.

+bot-info

Cette commande vous permet d'avoir différentes informations concernant DiscordBot.Js.

+channel-info

Cette commande vous permet d'afficher des informations concernant un salon.

+role-info

Cette commande vous permet d'afficher des informations concernant un rôle.

+server-list

Cette commande vous permet d'avoir la liste complète des serveurs connectés à DiscordBot.Js.

+server-invite

Cette commande vous permet de crée un lien d'invitation pour votre serveur et faire de la pub pour votre serveur.

+kick

Cette commande vous permet de kicker des membres de vos serveurs.

Dés que le membre est kicker, il reçoit un message disant qu'il a était kicker du serveur.

+ban

Cette commande vous permet de bannir des membres de vos serveurs.

Dés que le membre est kicker, il reçoit un message disant qu'il a était bannie du serveur.

+report

Cette commande vous permet de reporter des membres de vos serveurs.

Dés que le membre est kicker, il reçoit un message disant qu'il a était reporter du serveur.

+mute

Cette commande vous permet de muter des membres de vos serveurs.

Dés que le membre est kicker, il reçoit un message disant qu'il a était muter du serveur.

+unmute

Cette commande vous permet de démuter des membres de vos serveurs.

Dés que le membre est kicker, il reçoit un message disant qu'il a était démuter du serveur.

+bot-vote

Cette commande vous permet de votez sur Top.gg DiscordBot.Js.

+rps

Cette commande vous permet de jouez avec le bot à chifoumi.

+clear

Cette commande vous permet d'effacer de nombreux message (Limite 100 message).

+ping

Cette commande vous permet d'afficher des informations concernant la latence avec l'api et le serveur.

+say

Cette commande vous permet de faire parler DiscordBot.Js.

+say-markdown

Cette commande vous permet d'utilisez les markdown de discord facilement.

+logs-channel

Cette commande vous permet de configurer le salon des logs.

+setup-server

Cette commande vous permet de configurer vos serveurs facilement et rapidement.

+embed-help

Cette commande vous permet d'avoir la liste complète des commandes pour crée un embed.

+poll-help

Cette commande vous permet d'avoir la liste complète des commandes pour crée un sondage.

+xp-help

Cette commande vous permet d'avoir la liste complète des commandes pour le système de niveaux.

+music-help

Cette commande vous permet d'avoir la liste complète des commandes pour écouter de la musique.

+new-prefix

Cette commande vous permet définir un nouveau préfix.

+money-help

Cette commande vous permet d'avoir la liste complète des commandes pour le système d'argents.

+webhook-help

Cette commande vous permet d'avoir la liste complète des commandes pour crée un webhook.

+add-role

Cette commande vous permet de vous attribuez un rôle.

+remove-role

Cette commande vous permet de vous enlevez un rôle.

+unban

Cette commande vous permet d'avoir la liste complète des commandes de DiscordBot.Js.

+news

Cette commande vous permet d'avoir les actualités de DiscordBot.Js sur vos serveurs.
